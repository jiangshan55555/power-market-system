## 📊 研究设计与数据描述

### 1.1 研究框架

本研究采用**准实验设计**（Quasi-experimental Design），通过历史数据构建预测模型，并在模拟环境中验证投标优化策略的有效性。研究遵循**CRISP-DM**（Cross-Industry Standard Process for Data Mining）方法论，确保分析的科学性和可重复性。

### 1.2 数据集描述

**时间序列特征**：
- **观测期间**：2025年5月1日 00:30:00 至 2025年6月18日 23:45:00
- **总观测数**：N = 5,950个时间点
- **采样频率**：Δt = 15分钟（高频时间序列）
- **训练集**：n_train = 4,760（80%）
- **测试集**：n_test = 1,190（20%）

**变量定义**：
- **因变量**：y_t ∈ ℝ⁺（实时出清电价，CNY/MWh）
- **自变量**：X_t ∈ ℝ⁵（5维特征向量）
- **滞后约束**：严格遵守GAP规则，τ = 1

### 1.3 描述性统计分析

**目标变量统计特征**：
```
电价分布统计量
├── 均值 (μ): 395.48 CNY/MWh

├── 偏度 (Skewness): 0.127 (轻微右偏)

#### 2.1.1 模型理论基础

线性回归模型基于**高斯-马尔可夫定理**，在满足经典假设条件下，普通最小二乘估计量（OLS）具有**最佳线性无偏估计量**（BLUE）性质。

```
y_t = β₀ + β₁X_{t,1} + β₂X_{t,2} + β₃X_{t,3} + β₄X_{t,4} + β₅X_{t,5} + ε_t
```

其中：ε_t ~ i.i.d. N(0, σ²)

### 2.2 随机森林模型分析

#### 2.2.1 模型理论基础

随机森林基于**Bagging**（Bootstrap Aggregating）和**随机子空间方法**，通过集成多个决策树降低方差，提高泛化能力。理论上，当树的数量趋于无穷时，泛化误差收敛到：

**泛化误差上界**：
```
PE* ≤ ρ̄ × (1 - s²) / s²
```

其中：ρ̄为树间平均相关系数，s为单树强度

#### 2.2.2 模型配置与统计特征

**超参数设置**：
- **树的数量**: B = 200
- **每次分裂考虑特征数**: m = ⌊√p⌋ = 2
- **最小分裂样本数**: min_samples_split = 2
- **最小叶子样本数**: min_samples_leaf = 1

**Bootstrap统计分析**：
- **袋外样本比例**: 约36.8%（理论值：1 - (1-1/n)ⁿ ≈ 0.368）

#### 2.3.1 模型理论基础

**收敛速率**：
```
E[L(F_M)] - E[L(F*)] ≤ O(exp(-cM))
```

其中：c > 0为收敛常数，M为迭代次数

Obj = Σᵢ l(yᵢ, ŷᵢ) + Σₖ Ω(fₖ)

**随机优化模型**：
```
max E[U(Π(P_DA, λ_DA, λ_RT))]
s.t. P_DA ∈ [0, P_max]
