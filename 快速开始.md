## 🚀 三步部署到 Vercel

### 方法一：使用 Vercel CLI（最快 - 推荐）

```bash
# 1. 安装 Vercel CLI
npm install -g vercel

# 2. 登录
vercel login

# 3. 部署
cd power-market-system
vercel --prod
```

**完成！** 30 秒后获得在线 URL！

### 方法二：通过 GitHub

```bash
# 1. 创建 Git 仓库
cd power-market-system
git init
git add .
git commit -m "电力市场预测系统"

# 2. 推送到 GitHub
git remote add origin https://github.com/YOUR_USERNAME/power-market-system.git
git push -u origin main

# 3. 在 vercel.com 导入项目
# 访问 https://vercel.com → New Project → 选择仓库 → Deploy
```
